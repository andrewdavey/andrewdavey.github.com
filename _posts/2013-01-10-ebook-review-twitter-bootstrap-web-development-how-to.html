--- title: eBook Review - Twitter Bootstrap Web Development How-To layout: post ---

I was recently given a copy of a Twitter Bootstrap Web Development How-To to review.

The book provides an introduction to Twitter Bootstrap. Its step-by-step style walks the reader through the basics of getting started. You'll learn how to create and customize a variety of different page layouts and enhance these using some of Bootstrap's built-in jQuery plugins.

If you're new to web development and find online documentation a bit terse when learning new libraries, then this book is worth a quick read.
